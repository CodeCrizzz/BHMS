📌 Boarding House Management System

    The Boarding House Management System is a system designed to help manage and organize boarding house operations efficiently. It allows administrators to handle tenant information, room assignments, rental fees, and payment records in a centralized platform.

    This project aims to simplify manual record-keeping, reduce errors, and improve overall management of boarding house data. It was developed as a student project to apply concepts in database management, CRUD operations, and basic system design.

🛠️ Features:
    - Manage tenant information
    - Room and occupancy management
    - Rental and payment tracking
    - Organized record-keeping
    - User-friendly interface

💻 Technologies Used
    PHP
    MySQL
    HTML
    CSS
    JavaScript

🎯 Purpose of the Project

    This system was created for educational purposes to practice:
       - Database design and management
       - CRUD functionality
       - System analysis and development
       - Basic web application development
